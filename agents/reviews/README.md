# Reviews agents — local only

> Ce dossier n'est pas versionné sur Gitea.
> Il contient les captures d'outputs bruts des agents testés en conditions réelles.

## Convention

```
reviews/<Projet>/<agent>-v<N>.md
```

- `v1` = premier test réel
- `v2` = après amélioration de l'agent → comparo avec v1

## Boucle de feedback

Output capturé ici → gaps identifiés → agent amélioré (recruiter) → changelog mis à jour → re-test → v2
